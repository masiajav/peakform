# Replaid Lab — 15 piezas fuertes para Codex

Este paquete contiene 15 archivos `.md`, uno por pieza pilar.

## Instrucción para Codex

Usa cada archivo como especificación editorial y SEO. Crea o actualiza la página correspondiente manteniendo el diseño actual de Replaid Lab. Cada pieza debe resolver una pregunta concreta de un jugador de Overwatch, con ejemplos prácticos, estructura clara, FAQs, enlaces internos y CTA natural hacia guías relacionadas o expertos.

## Reglas globales

- Tono: claro, gamer, práctico, español natural.
- Evitar texto genérico tipo plantilla.
- Cada pieza debe tener Title SEO, meta description, slug, H1 único, intro, resumen rápido, ejemplos, checklist, FAQ y enlaces internos.
- Usar lenguaje prudente cuando dependa del parche o meta.
- No prometer subida de rango garantizada.
- Añadir `FAQPage` schema si el stack lo soporta.
- Añadir `Article` o `HowTo` schema cuando encaje.
- Añadir breadcrumbs.
